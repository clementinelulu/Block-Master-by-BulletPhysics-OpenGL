环境要求
操作系统
Windows 10/11（x64）
依赖库仅适配 Windows 平台（基于 _WIN32_WINNT 宏定义）

编译工具
Visual Studio 2019/2022（推荐），需安装「桌面开发使用 C++」组件
C++17 及以上标准
支持 OpenGL 3.3 及以上的显卡（需安装最新显卡驱动）

依赖库安装与配置
核心依赖库
GLFW：用于窗口创建与输入处理，推荐版本 3.3.x
GLEW：用于 OpenGL 扩展管理，推荐版本 2.1.0+
Bullet Physics：用于物理模拟（碰撞检测、重力效果），推荐版本 3.24+
Assimp：用于 3D 模型（OBJ/FBX 格式）导入，推荐版本 5.2.0+
GLM：用于矩阵、向量等数学计算，推荐版本 0.9.9.8+
SOIL：用于纹理图片加载，版本 1.16
ImGui：用于图形化 UI 界面开发，推荐版本 1.89+

链接器输入添加以下库文件：opengl32.lib、glfw3.lib、glew32.lib、glew32s.lib、BulletDynamics.lib、BulletCollision.lib、LinearMath.lib、assimp.lib、SOIL.lib、imgui.lib、imgui_impl_glfw.lib、imgui_impl_opengl3.lib

积木搭建大师 游戏说明书

游戏简介

欢迎来到积木搭建大师！这是一款基于物理引擎的3D积木搭建游戏。在一个温馨的房间环境中，使用6种不同形状的积木发挥你的创造力，搭建出各种有趣的结构。游戏采用真实物理模拟，让你体验真实的积木搭建乐趣！

游戏目标
主要目标: 使用积木搭建各种创意结构
评分目标: 尝试生成更多积木来获得更高分数和等级
创造目标: 发挥想象力，建造独特的积木作品

操作控制

🖱️ 摄像机控制
| 操作 | 功能 |
|------|------|
| `W` `A` `S` `D` | 前后左右移动摄像机 |
| `空格键` | 摄像机向上移动 |
| `Ctrl` | 摄像机向下移动 |
| `右键拖拽` | 旋转视角 |
| `Shift + WASD` | 快速移动（冲刺模式） |

🧱 积木生成
| 按键 | 积木类型 | 形状特点 |
|------|----------|----------|
| `1` | 拱形(Arch) | 经典拱门形状 |
| `2` | 圆锥(Cone) | 尖顶圆锥体 |
| `3` | 立方体(Cube) | 标准正方体 |
| `4` | 圆柱(Cylinder) | 圆柱形积木 |
| `5` | 三角柱(Prism) | 三角截面柱体 |
| `6` | 长方体(Cuboid) | 长方形积木 |

或者点击屏幕底部工具栏的积木图标

🔧 积木操作
| 操作 | 功能 |
|------|------|
| `左键点击积木` | 选择积木（变为黄色高亮） |
| `左键点击空白` | 取消选择 |
| `鼠标拖拽` | 水平移动选中的积木 |
| `T` | 向上移动选中的积木 |
| `B` | 向下移动选中的积木 |
| `R` | 重置选中积木的旋转（修正倒下的积木） |

🎮 游戏控制
| 操作 | 功能 |
|------|------|
| `ESC` | 退出游戏 |
| 点击 `START GAME` | 开始新游戏 |
| 点击 `END GAME` | 结束当前游戏并查看分数 |
| 点击 `NEW GAME` | 重新开始游戏 |

🖥️ 用户界面

📱 主控制面板（右上角）
- 开始界面: 显示 START GAME 按钮
- 游戏中: 显示当前分数、等级、操作提示和 END GAME 按钮
- 结束界面: 显示最终成绩和 NEW GAME 按钮

🛠️ 积木选择工具栏（底部）
- 仅在游戏进行时显示
- 包含6个积木图标按钮
- 悬停显示积木名称和对应按键
- 点击图标等同于按对应数字键

🏆 分数弹窗
游戏结束时显示详细评价：
- 总积木数量
- 最终等级
- 等级评语

 📊 评分系统

🎯 计分规则
- 每生成一个积木 = +1分
- 总分无上限
- 重新开始游戏时分数与积木清零

🏅 等级评定
| 分数范围 | 等级 | 评语 | 颜色 |
|----------|------|-------|------|
| 0-9分 | C级 | Keep practicing! | 🟡 黄色 |
| 10-19分 | B级 | Good! | 🔵 蓝色 |
| 20分以上 | A级 | Excellent! | 🟢 绿色 |

🔄 游戏流程

1. 启动游戏 → 看到主菜单界面
2. 点击 START GAME → 进入游戏，分数清零，清空旧积木
3. 选择积木 → 使用数字键或工具栏图标
4. 搭建创作 → 移动、堆叠积木，发挥创意
5. 点击 END GAME → 查看最终分数和等级
6. 点击 NEW GAME → 开始新一轮挑战，原积木清零

🔧 操作技巧
- 选择积木: 选中的积木会变成黄色，便于识别
- 精确放置: 使用T/B键进行精确的垂直定位
- 修正倾倒: 用R键快速修正倒下或倾斜的积木
- 重新开始: 每次开始新游戏都会清空所有旧积木
